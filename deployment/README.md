# Tourism Package Purchase Predictor

A Streamlit app that predicts whether a customer will purchase a tourism package,
powered by an XGBoost model registered on the Hugging Face Model Hub.

## Usage
Fill in the customer profile fields and click **Predict**.

## Model
Repository: `rknv1984/tourism-project-model`  
Algorithm: XGBoost (GridSearchCV-tuned, 5-fold CV)
